## 音韻論
螺語には母音音素が33、子音音素が46あるとされる。それぞれ以下のように分類される。

[目次に戻る](#contents)

<a id ="vowels"></a>
### 母音
母音音素は中枢母音音素32個と緩衝母音音素1個に分類される。

[目次に戻る](#contents)

<a id ="variable_vowels"></a>
#### 中枢母音
中枢母音は以下の4つに分類される。

- 前舌母音 (第1群)
- 前舌鼻母音 (第2群)
- 後舌母音 (第3群)
- 後舌鼻母音 (第4群)

またそれぞれに以下の4つの短母音と長母音が存在するので、一つの群には8つの母音音素が存在することになる。
よって中枢母音音素は32個である。

- 広母音
- 半広母音
- 非円唇狭母音
- 円唇狭母音

#### 補助子音
補助子音とは以下の子音の総称である。

<a id ="harmony"></a>
### 調和
螺語の形態論においては母音及び子音の調和が非常に大きな役割をもつ。

[目次に戻る](#contents)

<a id ="rule1"></a>
#### 中枢音規則
「実際に発話される任意の語は、音素として中枢音(中枢母音及び中枢子音)を1つ以上含んでいなければならない」という規則のこと。
例えばjEやmELなどの語形は中枢母音も中枢子音も含んでいないため許容されない。
一方でjamやZEjなどの語形は中枢母音と中枢子音の両方を持つわけではないが、中枢音を持ちはするので許容される。

[目次に戻る](#contents)

<a id ="rule2"></a>
#### 母音調和規則と子音調和規則
「実際に発話される任意の語において、その中枢母音はすべて同じ群のものでなければならない」「実際に発話される任意の語において、その中枢子音はすべて同じ群のものでなければならない」という規則のこと。2つを総称して調和規則ともいう。
そして実際の群の選択のことを調和という。例えばb^omHu:D^という語は母音が後舌母音(第3群)、子音が有声破擦音(第6群)で統一されているため、「3-6の調和をもつ」と表現される。

[目次に戻る](#contents)

<a id ="theoretical_word_forms"></a>
#### 仮想語形
仮想語形という概念は螺語文法の根幹をなす。以下ではそれを例によって説明する。

Tipa:ce「誰」とZuvQ:Io「こんにちは」は勿論異なる語形である。まず調和が異なり、前者は1-1の調和、後者は3-4の調和を持つ。しかしながらよくこの語を見ると「番号3の子音+番号3.の母音+番号1の子音+番号1:の母音+番号4の子音+番号2.の母音」の形であることには変わりないことがわかる。様々な理由からこれらを同じ語として扱う方が都合がよい。
その立場においてはTipa:ceもZuvQ:Ioも同じ語が別の調和をとっていると表現することができるが、その仮想的な語の語形は33.11:42.であると言える。これを「Tipa:ceとZuvQ:Ioはそれぞれ仮想語形33.11:42.の1-1の調和及び3-4の調和での実現である」などという。

## 品詞分類・形態論

[目次に戻る](#contents)

<a id ="content_words"></a>
### 内容詞
文の内容を中心的に表す語。語には中枢音のみが現れ、仮想語形はすべて番号で表される。
その調和によって次の「各調和の用法」の節で示すように文中での役割が表される。

e.g.) 33.11:42.「人」

[目次に戻る](#contents)

<a id ="affixes"></a>
### 包囲詞
内容詞に接頭辞や接尾辞あるいは接周辞としてついて意味を補足する語。語には中枢音と補助子音のみがそれぞれ1つ以上現れ、つく内容詞と同様の調和を示す。
仮想語形は番号と補助子音のみで表され、接頭辞として働くものは (包囲詞)= と、接尾辞として働くものは =(包囲詞) と、接周辞として働くものは (包囲詞)=(包囲詞) と書かれる。内容詞と包囲詞の境界も=で表し、=の周囲の環境に対応し緩衝音(緩衝母音あるいは緩衝子音)として発音される。

e.g.) 1.3H=3「その(抽象的な)場所にいることを表す」→1.3H=33.11:42.=3「社会の中で」

[目次に戻る](#contents)

<a id ="harmonical_affixes"></a>
#### 破格詞
ときには調和の表す役割を保存しながらも内容詞の調子を変えたい時がある。そのようなときに本来の調和を包囲詞で明示しながら調和を変えることがある。
このような調和の変更を破格といい、それに用いる包囲詞を**破格詞**という。特に母音第1群・子音第6群の調和(1-6の調和などという)への破格は特別で、文中での最も強い強調を表す。
これを強調の破格という。破格詞は以下の表のとおりである。

|  | 母音第1群 | 母音第2群 | 母音第3群 | 母音第4群 |
|:---:|:---:|:---:|:---:|:---:|
| 子音第1群 | 3.m2.= | 51.r= | =N3:2 | H3:6= |
| 子音第2群 | L2.=m1.2 | q3.= | N2:=4 | =2.n |
| 子音第3群 | r2.= | H1.=r3:4 | m1:=3.q | 3:q4= |
| 子音第4群 | 4L= | q3.6=42:N5 | 1.L3:2r= | N2:q= |
| 子音第5群 | r2.=m3:1 | 1L= | m3:= | N2.L5= |
| 子音第6群 | × | n3.L= | =12.mH3:3 | r3.=51:H |

<a id ="harmony_of_loan_words"></a>
##### 外来語の調和
外来語は語頭に一番近い(中枢母音に相当する音の)母音と(中枢母音子音に相当する音の)子音によって調和を判断する。例えばリパライン語のyuesleone「ユエスレオネ連邦」は、母音はu、子音はsをとって3-3の調和と見做す。
文中で外来語を使うときは、文中の役割を破格詞で明示して、その外来語の調和を用いる。例えばyuesleoneは1-1の調和で用いられる場合は`*jietleene`ではなく、`umo=yuesleone`と表す。

| 子音第2群 | `La~S=` | `sa~=me~` | `Ci^m=` | `me~s=` |
| 子音第3群 | `Xe^=a~r` | × | `ma~=Xe^n` | `a~Hx=` |
| 子音第4群 | `=mi^S` | `La^=xe~` | `y^r=S` | `e^r=xi~` |
| 子音第5群 | `Ly^=Xa~` | `i~f=he~` | `a^C=Hy~x` | `i^=Se~m` |

<a id ="interjections"></a>
### 遊離詞
文の内容と関係なく感情を表したり語調を整えたりする語。語に現れる音には調和以外の制限はなく、調和を変えられることも一切ない。
そのため仮想語形は実際の語形と同じである。文の切れ目には口調を調節する遊離詞を置くことが多いため句読点は用いない。
e.g.) o:X「嗚呼」

<a id ="semantics"></a>
## 各調和の用法
内容詞・包囲詞の調和は上でも述べたように「1-6の調和」などと呼ぶ。これは中枢母音は第1群、中枢子音は第6群のもので調和しているという意味であった。内容詞はこの調和によって大きく文中での役割を変える。

[目次に戻る](#contents)

<a id ="topical_harmony"></a>
### 焦点調和
中枢母音を第1群とする調和は、総じてその単語が文の焦点になることを意味するので「焦点調和」という。

[目次に戻る](#contents)

<a id ="te_harmony"></a>
#### 1-1の調和 (文の焦点についての疑問点)
1-1の調和は、ある文の焦点たりうる内容(1-2～1-5の調和で表されるような内容)を表す単語が文において疑問点としてあることを表す。注目されている疑問点というだけのことなので、この調和では文中の論理的意味関係を明確にするための包囲詞を用いるのが普通である。

`ta:T oH=W^t^Q~T^Q^=k^Q^mQ~`<br>

## 螺日辞典について
螺日辞典とはこのリポジトリで[公開している](https://github.com/SkurlavenijaMavija/rkharvanur/blob/master/%E8%9E%BA%E6%97%A5%E8%BE%9E%E5%85%B8.json)ラーヴァヌー語のOTM-JSON形式の辞書である。特殊な例外(よく知られたフレーズなど)を除いて仮想語形を見出し語とする。以下では螺日辞典の基本的な見方を説明する。

### 見出し語タグの一覧
螺日辞典の見出し語にはタグがついている。そのタグには以下のものがある。

+ 品詞タグ: "内容詞", "包囲詞", "遊離詞"の3つ。任意の見出し語について高々1つしかつかない。
+ 準品詞タグ: "破格詞", "集約詞", "補足詞"の3つ。このタグをもつ見出し語は必ず"包囲詞"タグをもつ。任意の見出し語について高々1つしかつかない。
+ 超品詞的タグ: "成句"の1つ。このタグをもつ見出し語には品詞タグや準品詞タグはつかない。

### 訳語タグの一覧
螺日辞典では訳語にもタグがついている。そのタグには以下のものがある。

+ 品詞タグ: "内容詞", "包囲詞", "遊離詞"の3つ。
+ 準品詞タグ: "破格詞", "集約詞", "補足詞"の3つ。
+ 超品詞的タグ: "成句"の1つ。

見出し語タグについているタグを、超品詞的タグ>準品詞タグ>品詞タグの順の優先度で訳語タグとして採用する。

### 内容項目の一覧
螺日辞典では見出し語の説明する項目が複数設けられていることがある。その項目には以下のものがある。

+ 基本義: 螺語の単語は調和によって非常に多様な意味を表すため、その用法の中核になる概念の把握が重要である。それを螺日事典では「基本義」項目として各見出し語の説明の最初に置いている。
+ 注意: 特殊な転義などを説明している。
+ 調和の用例: ある調和での見出し語の意味が学習者にとって注意が必要であると考えられるときに、それを注意するための項目である。`[稀]`はその調和での使用例がほとんど見受けられないことを示す。
+ 用例: 用例である。

### 見出し語参照事由一覧
螺日辞典では他の見出し語を参照するように促すことがある。その事由には以下のものがある。

+ 派生: 派生関係にある語を示す。
+ 類義語: 類義語を示す。
